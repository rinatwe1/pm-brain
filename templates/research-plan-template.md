## Synthesis Plan

### How to Organize Findings
1. **After each interview:**
   - Take notes in `interviews/interview-[N].md`
   - Tag key insights
   - Identify patterns

2. **After competitor analysis:**
   - Document in `competitors/[competitor-name].md`
   - Screenshot key features
   - Note what works / what doesn't

3. **Final synthesis:**
   - Compile all insights into `insights.md`
   - Create user personas
   - Document pain points
   - Identify opportunities
